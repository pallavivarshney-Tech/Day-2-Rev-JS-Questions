Question 3

Difference between var,let and const?

we can declare the variable name in one line or in the same line itself. Two variables with same names aren't possible in let. we can reassign/update the value if already asssigned. Let is a block-scoped. Let doesn't supports hoisting.

Hoisting - it brings the variables and functions at the top of thecode it supports only variable declaration but not its initialisation.

Const- we have to declare and initialise the variable name in same line. Two variables with same names aren't possible in const. we can't reassign/update the value if already initialised. Let is a block-scoped. Let doesn't supports hoisting.

Var- we can declare the variable name in one line or in the same line itself. Two variables with same names are possible in var. we can reassign/update the value if already asssigned. Let is a global-scoped. Var supports hoisting.

Question 4

Difference between “ == “ and “ === “ operators?

== double equals to checks only only for values not the data types but === triple equal checks for both the values as well as data type.

Question 5

What is an temperal dead zone ?

the term to describe the state where variables are un-reachable. They are in scope, but they aren't declared. The let and const variables exist in the TDZ from the start of their enclosing scope until they are declared.
